== Description ==
Tema de WordPress para Agencia Pum - Biblioteca de Libros

El objetivo de esta prueba técnica es evaluar la capacidad para crear un tema de WordPress desde cero e integrarlo con una maqueta HTML predefinida. El tema constará de una página principal que mostrará un vídeo y un texto, ambos gestionables desde el administrador de WordPress. También incluirá una rejilla de elementos, en este caso, libros, con un enlace desde cada elemento a una ficha individual. No se requiere un menú ni múltiples idiomas; la idea es mantenerlo simple para demostrar el entendimiento de la estructura de WordPress y cómo trabajar de manera eficiente y limpia.

Los libros serán un tipo de contenido personalizado (Custom Post Type) y tendrán campos personalizados, para lo cual utilizaremos Advanced Custom Fields (ACF). Los campos para cada libro son:

Título: Se utilizará el campo 'title' estándar de WordPress.
Autores: Campo repeater para poder incluir más de un autor.
Portada: Imagen de portada del libro.
Año de publicación: Información sobre el año en que fue publicado.
Género: Taxonomía propia del CPT "libro".
Para la base del tema, se puede utilizar una plantilla en blanco como BlankSlate o desarrollarla desde cero.

Inglés:

WordPress Theme for Pum Agency - Book Library

The aim of this technical test is to assess the ability to create a WordPress theme from scratch and integrate it with a predefined HTML template. The theme will consist of a homepage displaying a video and text, both manageable through the WordPress admin. It will also include a grid of elements, in this case, books, with a link from each element to an individual details page. No menu or multiple languages are required; the goal is to keep it simple to demonstrate understanding of WordPress structure and efficient, clean coding practices.

Books will be a Custom Post Type, and custom fields will be implemented using Advanced Custom Fields (ACF). The fields for each book include:

Title: Using the standard 'title' field in WordPress.
Authors: A repeater field to accommodate multiple authors.
Cover: Book cover image.
Publication Year: Information on the year of publication.
Genre: Custom taxonomy for the "book" CPT.
For the theme's foundation, a blank template like BlankSlate can be used, or it can be developed from scratch.
